# DS223 Marketing Analytics - Group Project Demo

## Product Overview

**Product Name:** Bayesian Carousel Bandits  
**Course:** DS223 Group Project  

This project demonstrates a marketing analytics platform that uses a microservice architecture to connect a database, backend API, frontend app, and supporting data science logic.

---

## Problem Definition

Marketing teams often need to decide which product, offer, or content alternative should be displayed to users. This project models carousel alternatives and interaction events so teams can test, track, and improve those decisions.

---

## Solution Architecture

**Microservice Components:**

- Frontend: Streamlit interface for interacting with the product.
- Backend: FastAPI service exposing endpoints for application logic.
- Database: PostgreSQL storage for carousels, alternatives, and interactions.
- Data Science: Python module for modeling or decision logic.
- Documentation: MkDocs and Quarto tutorial.
